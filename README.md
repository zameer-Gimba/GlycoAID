# 🩺 GlycoAID – Diabetes Risk Prediction System

## 📌 Project Overview
GlycoAID is a machine learning-powered web application designed to predict diabetes risk based on patient medical information.

The system assists healthcare workers by providing quick preliminary diabetes risk assessments using AI.

---

## 🚀 Features
- Predicts diabetes risk in real-time
- Uses machine learning classification
- Displays prediction probability
- Simple and clean Streamlit interface

---

## 🧠 Machine Learning Models Used
- Logistic Regression
- Decision Tree Classifier

The best-performing model was automatically selected based on accuracy.

---

## 📊 Dataset
Dataset loaded directly from GitHub:

https://raw.githubusercontent.com/plotly/datasets/master/diabetes.csv
